# cli-linter

A lightweight, non-strict JavaScript and Python linter written in Go. It suggests better ways to write code without enforcing fixes, focusing on developer-friendly guidance and clear, actionable messages.

## Features
- Language support: JavaScript, Python
- Opinionated but non-blocking suggestions
- Fast CLI written in Go
